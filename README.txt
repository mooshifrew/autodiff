Code Guide: 
network.py: network class code
layer.py: layer class code
config.py: datatypes
config.yaml: configuration file
model.py: Pytorch model definition
test_torch.py: custom model and pytorch model testing
utils.py: utility functions
activation.py: activation function classes code

test_network.ipynb: notebook testing implementation on supplied data
test-small-networks.ipynb: notebook testing implementation on small networks and comparing with tutorials
test-learning-rate.ipynb: notebook testing the effects of learning rate on training loss
compare_torch.ipynb: notebook for comparing our gradients with PyTorch for testing.
